Después de instalar ArchLinux configuro mi Vim así:

Descargá VIM plug:

curl -O https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

#### Aclaracion: Uso vim con tmux desde la TTY.

### Esto me dijo ChatGPT yo uso color pablo porque se ve bien cuando escribís mal una palabra.

```bash
export TERM=tmux-256color
echo $TERM
```

Dentro de .tmux.conf

```bash
# Colores correctos
set -g default-terminal "tmux-256color"

# Fuerza soporte de colores modernos
set -as terminal-overrides ",*:RGB"

# Evita problemas con aplicaciones de texto
set -g assume-paste-time 1

```


#### .vimrc:

```
" Activar colores
syntax on
set background=dark

" 256 colores
set t_Co=256

" True color (no rompe en TTY modernas)
set termguicolors
```
 

:echo &term debe decir: tmux-256color

:echo &t_Co debe decir 256

:hi Normal Si ves valores RGB → está perfecto.
