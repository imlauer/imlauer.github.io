Gemini is a super simple, text-focused alternative to HTTP—think of it as a minimalist web where you browse using special Gemini browsers like Lagrange, Amfora, or even command-line tools like AV-98 (which you can run in your terminal). Just install a Gemini browser, then visit sites using addresses that start with gemini:// instead of http://—no ads, no scripts, just pure content!

En Arch usé `amfora` para navegar por Gemeni.
```bash
yay -S amfora
```

* gemini.circumlunar.space: The main Gemini project hub, with guides and links.
* gemini://geminispace.info: A search engine for Gemini sites.
* gemini://tilde.team: A cozy community capsule with blogs and resources.
* gemini://warmedal.se/~antenna/: A Gemini news aggregator.
