```bash
git clone https://github.com/libthinkpad/apindex/
cd apindex
cmake . -DCMAKE_INSTALL_PREFIX=/usr
sudo make install
```

Luego ubicate en algún directorio y ejecutá:

```bash
cmake . -DCMAKE_INSTALL_PREFIX=/usr
sudo make install
```
