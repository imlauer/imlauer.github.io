Pasos a seguir para habilitar el modo compacto en Firefox 137+

```bash
about:config
browser.compactmode.show cambialo a true
```

Ahora andá al ícono de hamburguesa en tu navegador a la derecha superior.    
Seguí estos pasos:    

1. More Tools (Más herramientas)
3. Customize toolbar... 
4. Density
5. Compact (not supported)
6. Done
